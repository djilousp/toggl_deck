## Build containers :

```sh
make build_up
```

## Start Container:

```sh
make up
```

## Test:

```sh
make test
```

## Recreate Api container:

```sh
make recreate
```

## Prune All :

> :warning: **Local data might get lost**: This command will delete all containers, network and created volume !

```sh
make prune_all
```
